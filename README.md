# DesafioListaDeTarefas
O software é um gerenciador de tarefas que permite adicionar, concluir e listar atividades. O usuário vê uma lista numerada das tarefas pendentes, escolhe qual marcar como concluída e o programa continua ativo até que o usuário decida sair.

# Funcionalidades
Adicionar Tarefa: Permite ao usuário incluir novas tarefas na lista.
Marcar Tarefa como Concluída: Marca uma tarefa específica como concluída.
Visualizar Tarefas Pendentes: Mostra todas as tarefas pendentes e seu status atual.
Sair: Finaliza a aplicação.
